# Projeto de Qualidade de Dados

Projeto desenvolvido para prática de validações e controle de qualidade em datasets.

## Objetivo

Simular regras básicas de validação aplicadas em ambientes corporativos para garantir integridade e confiabilidade dos dados.

## Regras Implementadas

- Verificação de valores nulos
- Validação de formato de email
- Identificação de registros duplicados

## Tecnologias

- Python
- Pandas

## Resultado

Geração de relatório contendo inconsistências encontradas no dataset.# Qualidade-de-dados
